병원 회원가입 테스트 가이드

1. 테스트 URL

http://localhost:3002/register/hospital

2. 테스트 데이터

계정 정보

아이디: testhospital001
비밀번호: Test1234!
비밀번호 확인: Test1234!

기본 정보

대표자명: 홍길동
병원명: 테스트 동물병원
병원 설립일: 2020-01-01
사업자등록번호: 123-45-67890
연락처: 010-1234-5678
이메일: test@testhospital.com
웹사이트: https://www.testhospital.com
주소: (주소 검색으로 입력)

진료 정보

진료 가능 동물: 반려견, 고양이 선택
진료 분야: 내과, 외과 선택

사업자등록증

파일: PDF 또는 이미지 파일 선택
주의: 실제 파일을 업로드하여 S3 업로드 테스트

3. 테스트 확인 사항

  • 각 필드 유효성 검증이 작동하는지
  • 중복 확인 버튼들이 정상 작동하는지
  • 파일 업로드 시 "업로드 중..." 메시지가 표시되는지
  • PDF 파일 업로드가 정상적으로 완료되는지
  • 회원가입 완료 후 로그인 페이지로 이동하는지

4. 데이터베이스 확인

회원가입 완료 후 다음 테이블들을 확인:

  • users 테이블
  • hospital_animals 테이블
  • hospital_specialties 테이블
  • hospital_business_licenses 테이블
